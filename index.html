・Let's Code で学べる言語

HTML

HTMLはWebページを作成するための言語です。指定した文章、画像、URLなどを表示できます。

CSS

CSSはそのWebページをデザインします。HTMLで指定した要素の大きさを変えたり、色をつけれます。

Java Script

Java Script はHTMLやCSSで指定した要素に動きをつけます。

・ Let's Code を使う理由

1 各言語を隅々まで解説

Let's Codeは各言語を隅々まで解説しているので、各言語について深く学べます。高校で必修化されたプログラミングもLet's Codeで学べば困ることはありません。素早く学びたい人には、速習編があるので、自分に合った学び方ができます。

2 詳しい解説動画


入門編の動画を見てみる

Let's Codeは全ての講座に解説動画がついていて、どこを変えたら何が変わるのか、確認しながら学習できます。
読むだけでは理解できないところも、何度での見直せます。

3 講座→即演習で確実に書く力が身につく

Let's Codeは全ての講座の後にドリルがついているので、コードを書く力が確実に身につきます。
分からなかったところは、用復習ボタンで後から見返せるため、効率的に復習できます。

4 挫折しないプログラミング学習を実現するコードチェック

プログラミングでは一文字でもコードが違うとエラーが出てしまいます。そのため、何が間違えているか分からず挫折してしまう人が多くいます。しかし、コードチェックを使えば、どこが違うのか一目でわかるのでその心配はありません。
